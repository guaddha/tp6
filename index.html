    las variables como ejempllo las var son una función/local, mientras que las let y const tienen un funcionan como bloque y podemos declararlas de diferentes formas como por ejmeplo:

    • Function declaration

    • Function constructor
    • IIFE(immediately Invoked Function Expression)
